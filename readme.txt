====================================================== 純前端 PDF 編輯器 (Cloudflare Worker 版) 使用說明書
歡迎使用本 PDF 編輯器！本工具完全在您的瀏覽器前端運行，不會將您的機密文件上傳至任何後端伺服器，確保最高等級的隱私安全。
以下是各項功能的操作指南：
一、 基礎操作與工具列
在畫面上方，您會看到一排主要工具列：
1. 📂 開啟：點擊以上傳並載入您的 PDF 檔案（支援多頁）。
2. 👆 選取：這是最重要的模式！在此模式下，您可以點擊並拖曳移動畫面上的「文字」、「圖片」與「畫筆塗鴉」。
3. ✋ 移動：當畫面放大時，切換到此模式可按住滑鼠左鍵拖曳、平移視角。
4. ✍️ 畫筆：自由在 PDF 上塗鴉或簽名。
5. 📝 文字：選擇此工具後，在畫面的「任意空白處」點擊一下，即可產生新的文字方塊並開始打字。
6. 🖼️ 圖片：將電腦中的圖片 (JPG/PNG) 貼到 PDF 上。
二、 如何「修改」PDF 上原本就有的文字？
這是本編輯器的強大特色（立可白遮罩技術）：
1. 點選上方的「👆 選取」工具。
2. 將滑鼠移到 PDF 畫面上原本的文字，游標會變成文字輸入圖示。
3. 直接「點擊」該行文字，系統會自動在底層塗上白色立可白，並將該文字轉為可編輯的狀態！
4. 您現在可以自由刪除、修改裡面的內容，甚至將它拖曳到其他位置。
三、 調整物件屬性 (顏色、大小、圖層、刪除)
請務必先切換到「👆 選取」工具，並「點擊選中」畫面上的文字、圖片或塗鴉：
* 此時，上方工具列會亮起【屬性工具列】。
* 顏色與大小：您可以即時修改所選文字的顏色、粗體 (B)、斜體 (I)、字體大小；若是畫筆，則可修改線條粗細與顏色。
* 圖片縮放：選中圖片後，圖片右下角會出現「藍色圓點」，按住它拖曳即可等比例縮放圖片。
* ⏫ 置頂 / ⏬ 置底：若物件互相遮擋，點擊此按鈕可調整圖層前後順序（成功時物件會發出綠色閃爍光暈）。
* 🗑️ 刪除：點擊可刪除您新增的物件。如果您刪除的是「修改過的 PDF 原有文字」，它會自動復原成原本無法編輯的初始狀態。
四、 ⚠️ 重要：關於中文亂碼與「無法下載」的解決方案
由於 PDF 底層的歷史包袱，預設只支援英文字體。當您輸入「中文字」並點擊儲存時，系統會嘗試從雲端下載開源中文字體來合成檔案。
【如果點擊下載後，系統跳出錯誤或沒有反應？】 這通常是因為您的網路環境、公司防火牆、或是瀏覽器的「廣告阻擋器 (AdBlocker)」封鎖了雲端字體的下載，導致系統發生 "WinAnsi cannot encode" 的崩潰錯誤。
👉 終極解決方案（自訂字體防呆機制）：
1. 點擊工具列右側的黃色按鈕「🔤 自訂字體」。
2. 從您的電腦中，選擇一個中文字體檔上傳（副檔名為 .ttf 或 .otf）。
   * Windows 用戶通常可以在 C:\Windows\Fonts 資料夾中找到字體（例如微軟正黑體、標楷體）。
3. 載入成功後，再次點擊「💾 儲存下載」，保證 100% 成功匯出，且絕無亂碼！
五、 縮放與儲存
* 🔍- / 🔍+：放大或縮小文件以利於精細編輯（向量渲染技術保證放大絕對清晰）。
* ◀ / ▶：切換 PDF 的上一頁與下一頁。
* 💾 儲存下載：完成所有編輯後，點擊此按鈕。系統會將您的修改完美合成，並自動下載一份新的 PDF 檔案到您的電腦中。
祝您使用愉快！
